sm-photo-tool by jmrodri

CLI client to mange photos on smugmug.com

CLI client to manage photos and galleries on smugmug.com. Using sm-photo-tool requires an account on smugmug.com. sm-photo-tool is based on the original sm_tool.py (smugmug.py) by zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/jmrodri/sm-photo-tool

get the source code on GitHub : jmrodri/sm-photo-tool
